Desarrollar juego cachipun (piedra, papel o tijeras)
Que se pueda jugar contra el computador y se pueda indicar cuentas veces se desea jugar o repetir el juego.
Cada uno puede elegir la opcion de forma secreta y compararla, para determinar quien gana.
Reglas:
● Tijera le gana a papel
● Papel le gana a piedra
● Piedra le gana a tijera
● Si ambos jugadores eligen la misma opción es un empate

1) Solicitar cuantas veces debe repetirse el juego consecutivamente. Se deben mostrar los resultados y luego pedir respuesta segun lo que haya solicitado el jugador.

2) Solicitar jugada al usuario
● Piedra
● Papel 
● Tijera

3) Generar una jugada automatica de la maquina usando math.random() de js

4) Definir a un ganador considerando la jugada del usuario y la generada automaticamente para la maquina

5) Indicar el resultado de la partida dependiendo del caso
● Felicitar al ganador en caso de ser el usuario.
● Indicarle al usuario que ha perdido contra la máquina en caso de que ésta sea la que gane.
● Declarar un empate.
